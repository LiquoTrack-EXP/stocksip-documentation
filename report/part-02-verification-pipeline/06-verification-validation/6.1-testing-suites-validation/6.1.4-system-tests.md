### 6.1.4. Core System Tests
En esta subsección se presentan las pruebas de sistema realizadas sobre la aplicación web de StockSip, ejecutadas mediante Selenium IDE. Estas pruebas validan el comportamiento del sistema desde la perspectiva del usuario final, simulando flujos completos de interacción con la interfaz gráfica. A través de estas pruebas se verifica que los casos de uso críticos funcionen correctamente en un entorno real, asegurando la correcta navegación, validación de formularios y respuesta visual del sistema ante las acciones del usuario.
