#### 6.2.1.2. Code Quality & Code Security

La calidad del código y la seguridad son esenciales para el desarrollo confiable de la plataforma StockSip, especialmente considerando que el sistema gestiona información sensible de negocios como stock, pedidos y datos de usuarios.

- **Calidad del Código:** La calidad del código se mide mediante métricas como la cobertura de pruebas unitarias e integración implementadas en el proyecto, y la complejidad ciclomática de los componentes críticos como los servicios de gestión de inventario y autenticación. El equipo utiliza SonarQube para realizar un seguimiento continuo de la calidad del código en los repositorios de StockSip, asegurando que cumpla con los estándares establecidos. SonarQube proporciona un análisis exhaustivo del código fuente del backend en ASP.NET Core y la aplicación web en Vue, detectando problemas de calidad como duplicación de código, métodos con alta complejidad o dependencias innecesarias, y ofreciendo sugerencias concretas para mejorarlos.

- **Seguridad del Código:** Es crucial identificar y mitigar vulnerabilidades comunes en la plataforma StockSip, como inyecciones en las consultas a MongoDB, exposición de datos sensibles en los endpoints de la API RESTful o validaciones insuficientes en los formularios de registro y gestión de productos. La implementación de prácticas de codificación segura y la validación adecuada de las entradas del usuario son fundamentales para proteger tanto la información de los dueños de licorerías como la de los proveedores. El equipo utiliza SonarLint integrado en los IDEs de desarrollo, como Rider para el backend y VS Code para la aplicación web, lo que permite detectar problemas de calidad y seguridad en tiempo real mientras se escribe el código. Esta práctica fomenta una cultura de calidad y mejora continua desde el inicio del desarrollo, asegurando que los problemas sean identificados y resueltos antes de llegar a las etapas de revisión y prueba.
