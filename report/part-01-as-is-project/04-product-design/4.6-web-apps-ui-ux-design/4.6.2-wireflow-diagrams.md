### 4.6.2. Web Applications Wireflow Diagrams

Un wireflow o flujo de pantalla es un diagrama donde se reúnen distintos wireframes cuya finalidad es representar las metas del usuario dentro de la aplicación y el proceso mediante el cual las consigue. Los pasos para la creación de cada diagrama comienzan con la definición de un objetivo del usuario que desea cumplir. Luego, se define el flujo de tareas que deben ser realizadas por el usuario en la aplicación para conseguir dicho objetivo. Finalmente, se traducen dichas tareas en pantallas y se trazan las decisiones asociadas a los elementos interactivos del wireframe.
 
A continuación se presentan los wireflow diagrams correspondientes a los principales User Goals identificados para la aplicación web **StockSip**.
