### 4.2.5. Navigation Systems

La navegación en **StockSip** fue diseñada para facilitar el recorrido del usuario de manera clara, rápida e intuitiva, tanto en la landing page como en la aplicación web. La estructura de navegación busca garantizar un acceso ordenado a las secciones y funcionalidades principales de la plataforma.

#### Landing Page

En la landing page se implementó una barra de navegación fija con las secciones clave para el visitante, permitiendo un desplazamiento más ágil entre los contenidos informativos del sitio. Las secciones incluidas son las siguientes:

- Inicio
- Producto
- Características
- Beneficios
- FAQ
- Contacto
- Empezar

#### Aplicación Web

En la aplicación web, se implementó una barra lateral fija con íconos, la cual permite el acceso directo a las funcionalidades clave del sistema. Esta navegación ha sido organizada para facilitar el uso continuo de la plataforma sin interrumpir el contexto de trabajo del usuario.

Las principales secciones disponibles son:

- Inicio
- Almacenes
- Guías de Cuidado
- Pedidos
- Ventas (Prov.)
- Catálogos
- Productos
- Proveedores
- Carrito
- Suscripciones
- Panel de Administración
- Perfil

Cada sección está representada por un ícono claro y una etiqueta visible al pasar el cursor o al expandir el menú, lo que contribuye a una navegación fluida, comprensible y eficiente dentro del sistema.
