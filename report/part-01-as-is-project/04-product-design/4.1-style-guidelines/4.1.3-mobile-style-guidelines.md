### 4.1.3. Mobile Style Guidelines
StockSip adopta un enfoque de diseño unificado (cross-platform), utilizando una misma base visual y de interacción para Android e iOS. Esto permite mantener consistencia en la experiencia del usuario, reducir costos de desarrollo y facilitar el mantenimiento.

No obstante, se realizan adaptaciones específicas para cada sistema operativo con el fin de respetar sus patrones nativos y mejorar la usabilidad.


#### 4.1.3.1. iOS Mobile Style Guidelines

En iOS, la interfaz mantiene el diseño general de StockSip, pero se adapta a las Human Interface Guidelines para garantizar una experiencia natural para el usuario.

**Adaptaciones principales**

- Navegación mediante **Navigation Bar** y **Tab Bar** en lugar de menú lateral  
- Deslizado integrado dentro de la interfaz para retroceder
- Componentes con estilo más plano (sin elevaciones pronunciadas)  
- Uso de gestos (swipe) como principal mecanismo de interacción  
- Feedback mediante **alerts** y **activity indicators**

#### 4.1.3.2. Android Mobile Style Guidelines

En Android, la interfaz utiliza la misma base visual, adaptándose a los principios de Material Design.

**Adaptaciones principales**

- Uso de **Navigation Drawer** para navegación principal  
- Inclusión de **Floating Action Button (FAB)** para acciones clave  
- Componentes con elevación (sombras) para jerarquía visual  
- Feedback táctil mediante efecto ripple  
- Uso de **Snackbar** para mensajes temporales
