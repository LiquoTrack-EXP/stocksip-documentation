### 5.2.5. Implemented Native-Mobile Application Evidence

Para el despliegue de esta aplicación, se utilizó Firebase App Distribution y Github Releases, que permiten distribuir la aplicación a los usuarios de prueba de manera eficiente. Firebase App Distribution es una plataforma que facilita la distribución de aplicaciones móviles a un grupo específico de usuarios, lo que permite recopilar comentarios y realizar pruebas antes del lanzamiento oficial. Por otro lado, Github Releases es una función de GitHub que permite a los desarrolladores publicar versiones específicas de su software, proporcionando un lugar centralizado para descargar la aplicación y acceder a notas de la versión.

Para el despliegue de la aplicación móvil en Android, se utilizó Github Releases y Firebase App Distribution, herramientas que permiten alojar archivos binarios directamente desde un repositorio de GitHub y distribuir aplicaciones móviles a través de Firebase.

Se adjunta el enlace para descargar directa de la aplicación móvil en Android desplegada en Github Realases: https://github.com/LiquoTrack/stocksip-android-app-distribution/releases/download/v.2.0.0/stocksip-app.apk
