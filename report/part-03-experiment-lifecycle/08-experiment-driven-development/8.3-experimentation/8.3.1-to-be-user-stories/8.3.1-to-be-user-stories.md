## 8.3. Experimentation

### 8.3.1 To-Be User Stories

| ID    | Título                            | Descripción                                                                                                                                                                       | User               |
|-------|-----------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------|
| US-01 | Crear catálogo de productos       | Como proveedor, quiero crear un catálogo de productos con información detallada de los artículos que ofrezco, para mostrar mi inventario disponible a las licorerías interesadas. | Proveedor          |
| US-02 | Publicar catálogo                 | Como proveedor, quiero publicar mi catálogo de productos, para que las licorerías puedan visualizar y evaluar mis ofertas comerciales.                                            | Proveedor          |
| US-03 | Editar catálogo                   | Como proveedor, quiero actualizar la información de mi catálogo, para mantener los precios, stock y productos disponibles siempre actualizados.                                   | Proveedor          |
| US-04 | Explorar catálogos de proveedores | Como dueño de licorería, quiero visualizar los catálogos publicados por distintos proveedores, para comparar productos y encontrar las mejores opciones de abastecimiento.        | Dueño de licorería |
| US-05 | Agregar productos al carrito      | Como dueño de licorería, quiero agregar productos de distintos catálogos a un carrito de compras, para consolidar los artículos que deseo adquirir antes de realizar un pedido.   | Dueño de licorería |
| US-06 | Comprar productos del carrito     | Como dueño de licorería, quiero confirmar la compra de los productos agregados al carrito, para abastecer mi negocio de manera rápida y organizada.                               | Dueño de licorería |
| US-07 | Visualizar el estado del pedido   | Como dueño de licorería, quiero conocer el estado de mi pedido, para estar informado sobre la entrega y planificar mis operaciones comerciales.                                   | Dueño de licorería |
