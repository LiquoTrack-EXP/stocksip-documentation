#### 8.3.3.4. Implemented To-Be Native-Mobile Application Evidence

Para esta sección, se presentan las evidencias las historias de usuario implementadas en la aplicación nativa móvil de StockSip, que reflejan las funcionalidades clave desarrolladas para los dueños de licorerías. Cada historia de usuario incluye capturas de pantalla y descripciones del objetivo alcanzado.

#### US-01: Visualización de Stock en los catálogos

Para la historia de usuario US-01, se implementó la funcionalidad que permite a los dueños de licorerías visualizar la disponibilidad de stock actual de los productos en los catálogos de los proveedores. La aplicación muestra claramente si un producto está disponible o agotado, evitando que los usuarios realicen pedidos de artículos no disponibles.

![https://i.imgur.com/Z1fdvzq.png](https://i.imgur.com/Z1fdvzq.png)

#### US-02: Crear dirección de entrega

Para la historia de usuario US-02, se desarrolló la funcionalidad que permite a los dueños de licorerías ingresar y registrar una dirección de entrega al realizar su pedido. La aplicación valida la dirección ingresada y asegura que el proveedor despache el producto al lugar correcto.

![https://i.imgur.com/IRbpRyl.png](https://i.imgur.com/IRbpRyl.png)

#### US-03: Explorar catálogos de proveedores

Para la historia de usuario US-03, se implementó la funcionalidad que permite a los dueños de licorerías explorar y comparar los catálogos publicados por distintos proveedores. La aplicación facilita la búsqueda de productos y la comparación de precios y disponibilidad, ayudando a los usuarios a encontrar las mejores opciones de abastecimiento.

![https://i.imgur.com/rkoKXyu.png](https://i.imgur.com/rkoKXyu.png)

#### US-04: Comprar productos del carrito

Para la historia de usuario US-04, se desarrolló la funcionalidad que permite a los dueños de licorerías confirmar la compra de los productos agregados al carrito. La aplicación proporciona un proceso de compra rápido y organizado, asegurando que los usuarios puedan completar sus pedidos de manera eficiente.

![https://i.imgur.com/IZGsZaV.png](https://i.imgur.com/IZGsZaV.png)

#### US-07: Visualizar el estado del pedido

Para la historia de usuario US-07, se implementó la funcionalidad que permite a los dueños de licorerías visualizar el estado de sus pedidos. La aplicación muestra información actualizada sobre el progreso del pedido, incluyendo confirmación de recepción, preparación, despacho y entrega, lo que permite a los usuarios planificar sus operaciones comerciales de manera efectiva.
